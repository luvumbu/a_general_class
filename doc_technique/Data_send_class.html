Documentation Technique de la Classe AsciiConverter

Description Générale

La classe AsciiConverter permet de convertir des chaînes de caractères en code ASCII et inversement.

Propriétés

  • $inputString : La chaîne de caractères à convertir.
  • $asciiArray : Tableau contenant les valeurs ASCII de la chaîne convertie.

Méthodes

__construct($inputString)

Constructeur de la classe. Initialise la chaîne de caractères à convertir.

Exemple d'utilisation :


$converter = new AsciiConverter("Bonjour");

convertToAscii()

Convertit la chaîne de caractères en tableau de valeurs ASCII.

Exemple d'utilisation :


$asciiArray = $converter->convertToAscii();

convertToString()

Convertit un tableau de valeurs ASCII en une chaîne de caractères.

Exemple d'utilisation :


$string = $converter->convertToString([66, 111, 110, 106, 111, 117, 114]);
